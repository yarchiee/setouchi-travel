# 瀨戶內海建築藝術之旅 - Travel App

A beautiful travel itinerary app for your 9-day Japan trip.

## Quick Start

### Install Node.js
First, install Node.js from https://nodejs.org/ (LTS version recommended)

### Install Dependencies
```bash
cd setouchi-travel
npm install
```

### Run Locally
```bash
npm run dev
```
Open http://localhost:5173 in your browser.

## Deploy to Vercel (Recommended - Free)

1. Push this folder to GitHub
2. Go to https://vercel.com
3. Sign in with GitHub
4. Click "Add New Project"
5. Import your repository
6. Click "Deploy"

Your app will be live at `https://your-project.vercel.app`

## Deploy to Netlify (Alternative - Also Free)

1. Push to GitHub
2. Go to https://netlify.com
3. Click "Add new site" → "Import an existing project"
4. Connect to GitHub and select your repo
5. Build settings are auto-detected
6. Click "Deploy"

## Build for Production

```bash
npm run build
```

The `dist` folder contains the production build.
